# Luo Multilingual Master Pack

This document strengthens Kaya AI for Luo-first and multilingual conversations.

## Core Luo Identity
- Dholuo is a Nilotic language spoken mainly in western Kenya and northern Tanzania.
- "Jaluo" refers to Luo people (singular), and "Jo-Luo" can be used for Luo people (plural/community usage).
- Common greeting style values respect, clarity, and relational tone.

## High-Frequency Luo Greetings and Responses
- Misawa? -> Misawa ahinya.
- Nango? -> Adhi maber.
- Idhi nade? -> Adhi maber.
- Oriti? -> Oriti maber.
- Erokamano -> Erokamano ahinya / Onge gima.
- Oyaore -> Oyaore ahinya.
- In nadi? -> An maber.
- Ber? -> Ber ahinya.

## Daily Conversation Snippets
- Adwaro kony mar somo. -> I need help with studying.

- An gi penjo achiel. -> I have one question.

## Luo Learning Mini Grammar
- Pronouns:
  - an = I
  - in = you
  - en = he/she/it
  - wan = we
  - un = you (plural)
  - gin = they
- Useful particles:
  - ka = if/when
  - to = but/then
  - nikech = because
  - mondo = so that
  - kata = or
- Negation patterns:
  - ok + verb
  - examples: ok anyal, ok awinjo, ok adwaro

## Multilingual Policy Notes for Assistant
- If user speaks Luo, answer in Luo unless asked otherwise.
- If user asks translation, provide both Luo and the requested target language.
- If user uses mixed Luo+English, preserve Luo tone and clarify technical terms in English where useful.
- Keep explanations simple first, then add detail if requested.

## Extra Luo Q&A Seeds
Q: Inyalo puonja Luo kuom step-by-step?
A: Ee. Anyalo chako gi greetings, penjo moting'o, to wabiro dhi nyime gi grammar kod vocabulary pile ka pile.

Q: Ka adwaro yudo tich mar teknoloji, atim nade?
A: Chak gi somo mar msingi e komputer, coding, kod internet safety. To ket portfolio, tim project matin matin, kendo penj jomoko ma en gi tich e sektoru.

Q: Inyalo loko weche ma togo e English?
A: Ee, anyalo. Chak ka ichiwo paragraph to anyalo loko e English kata e Luo kaka idwaro.

Q: Nitie yo mar puonjruok ma tek matin?
A: Ee. Pangwa saa matin pile, tim revision, kendo yud osiep mar somo mondo ukonyruok.

Q: Inyalo konyo gi website?
A: Ee. Anyalo chweyo website, miyo HTML/CSS/JS, kendo nyisi kido mar mobile-friendly design.
