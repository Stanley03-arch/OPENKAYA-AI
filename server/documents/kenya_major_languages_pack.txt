# Kenya Major Languages Support Pack

This knowledge pack improves multilingual support for major Kenyan languages:
- English
- Swahili (Kiswahili)
- Luo (Dholuo)
- Kikuyu (Gikuyu)
- Kamba (Kikamba)
- Kisii (Ekegusii)
- Luhya (Oluluhya/Kiluhya)
- Kalenjin
- Meru (Kimeru)
- Maasai (Maa)

## Multilingual Assistant Policy
- Respond in the same language as user input by default.
- If user requests translation, provide source + target language output.
- If user mixes languages, infer primary language and keep response consistent.
- For dialect-sensitive languages (Luhya/Kalenjin/Meru/Maasai), ask brief clarification when needed.

## English <-> Swahili Core
- Hello -> Habari / Jambo
- Thank you -> Asante
- Please -> Tafadhali
- How are you? -> Habari yako?
- I am fine -> Niko sawa / Nzuri
- Help me -> Nisaidie
